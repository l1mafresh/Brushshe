# Brushshe - програма для малювання

## Опис
Брашше - простий графічний редактор, написаний на Python 3 та CustomTkinter. В ньому можна малювати, змінювати розмір та колір пензля, змінювати колір тла, прикрасити малюнок наліпками (поки їх 5 штук). Є світлий та темний режим. В майбутньому додам більше функціоналу.

![Screenshot](https://github.com/l1mafresh/brushshe/blob/main/screenshot.png)

## Встановлення
1. Встановіть Python 3.11 (можливо, підійдуть і новіші чи старіші версії);
2. Встановіть потрібні бібліотеки, якщо не встановлені - відкрийте термінал чи командну строку та введіть ці команди:

```
pip install customtkinter
pip install CTkMenuBar
pip install CTkColorPicker
pip install ctkmessagebox
```

3. Запустіть Python IDLE, відкрийте файл brushshe.py та запустіть його.
